# Rumline

A macOS workspace for solo contractors juggling many client projects — project health that derives itself, capture from anywhere, and a daily brief that earns its place.

**Website:** https://rumlineapp.com

## About this repo

This is the public releases repo for Rumline. It hosts:

- **Release downloads** — published under [Releases](../../releases)
- **`appcast.xml`** — the [Sparkle](https://sparkle-project.org) feed the app uses to check for and deliver updates
- **Marketing site** — `index.html` and `styles.css`, served at [rumlineapp.com](https://rumlineapp.com)

The app source code lives in a separate private repository.

## Installation

Download the latest `Rumline-x.y.z.zip` from the [Releases page](../../releases/latest), unzip it, and move `Rumline.app` to your Applications folder.

**Requirements:** macOS 15.4 or later · Apple Silicon

## Updates

Rumline updates itself via Sparkle using `appcast.xml` in this repo — no need to revisit this page after installing.
